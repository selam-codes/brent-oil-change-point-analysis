# Notebooks

- `1.0-eda.ipynb` — Data loading, cleaning, trend/stationarity/volatility analysis (Task 1 & 2 EDA).
- `2.0-bayesian-change-point.ipynb` — PyMC Bayesian single change point model, convergence diagnostics, and event association (Task 2).

Run notebooks from the project root with the project's virtual environment (`.venv`) selected as the Jupyter kernel, so relative paths to `data/` resolve correctly.
